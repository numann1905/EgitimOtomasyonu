# 🎓 ÜSEM Eğitim Otomasyonu Projesi

Bu proje, bir eğitim merkezinin kurslarını yönetmek ve öğrenci kayıtlarını almak için geliştirilmiş bir web otomasyonudur.

## 🛠️ Kullanılan Teknolojiler
* **Backend:** Python (Flask)
* **Veritabanı:** SQLAlchemy (SQLite)
* **Güvenlik:** Bcrypt şifreleme
* **Frontend:** HTML5, Jinja2, Bootstrap

## 🚀 Kurulum ve Çalıştırma
Projeyi kendi bilgisayarınızda çalıştırmak için şu adımları izleyin:

1. Bu depoyu klonlayın: `git clone https://github.com/numann1905/EgitimOtomasyonu.git`
2. Proje klasörüne girin.
3. Gerekli kütüphaneleri yükleyin:
   ```bash
   pip install Flask Flask-SQLAlchemy bcrypt
